📊 Data Analytics Project: Customer Behavior Dashboard
📌 Overview

This project is an end-to-end data analytics workflow that transforms raw customer data into actionable business insights. It includes data cleaning, feature engineering, SQL-based querying, and an interactive Power BI dashboard.

🎯 Objective

To analyze customer behavior and identify key patterns in purchase activity, subscription status, and product preferences to support data-driven business decisions.

🛠️ Tools & Technologies
Python (Data Cleaning & Feature Engineering)
MS SQL Server (Data Storage & Querying)
Power BI (Data Visualization & Dashboarding)
📂 Project Workflow
1. Data Cleaning & Feature Engineering (Python)
Handled missing values
Performed basic feature engineering
Prepared dataset for database integration
2. Database Management (SQL Server)
Imported cleaned dataset into MS SQL Server
Executed SQL queries for analysis
Extracted meaningful business insights
3. Data Visualization (Power BI)
Built interactive dashboard with multiple visuals:
Customer segmentation (subscription status, gender, category)
Purchase behavior analysis
Age-based and category-based insights
Used DAX measures for KPIs:
Number of Customers
Average Purchase Amount
Average Review Rating
📊 Dashboard Features
Slicers:
Subscription Status
Gender
Category
Shipping Type
Visuals:
Customer distribution by subscription status
Sales and reviews by category
Age group analysis for purchases and ratings
KPI cards for key metrics
📈 Key Insights
Identified customer distribution across subscription types
Analyzed purchase behavior across different age groups
Evaluated category-wise sales performance
Measured customer satisfaction using review ratings
